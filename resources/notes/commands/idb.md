- List availables devices
    - `idb list-targets`
- Boot a simulator
    - `idb boot --udid <simulator-udid>`
- Install an App on a Device:      `
    - `idb install /path/to/your.app`
- Launch an App:
    - `idb launch <bundle-id>`
- Uninstall an App:
    - `idb uninstall <bundle-id>`
- Clear App Data:
    - `idb clear-app-data <bundle-id>`
